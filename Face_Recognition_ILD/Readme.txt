# Face Recognition - Quick Start

## How to Run

1. Install Requirements  

   pip install -r requirements.txt

2. Train the Model

   *(First-time setup)*  
   python src/train.py

3. Real-Time Recognition

   python src/real_time.py
   
   - Press `q` to quit  
   - Predictions appear in bounding boxes  

4. Test Performance(Optional)  
   
   python src/test.py
